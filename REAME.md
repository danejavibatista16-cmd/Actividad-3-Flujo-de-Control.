# Actividad 3: Flujo de Control (Parte 2)

**Asignatura:** Programación I  
**Profesor:** Gamalier Reyes del Carmen  

## Resumen del Proyecto
Este programa en C++ está diseñado para gestionar el flujo de calificaciones de una cantidad `n` de estudiantes. Utiliza bucles `for` para la captura dinámica de datos y estructuras condicionales `if-else` para determinar si el alumno ha aprobado o reprobado basándose en un promedio mínimo de 70 puntos. La salida de datos implementa la librería `<iomanip>` (`setw`) para garantizar un diseño tabular limpio, alineado y legible en la consola.
